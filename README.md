# Electronics / 电路设计

[简体中文](#简体中文) | [English](#english)

## 简体中文

### 简介
本目录包含 EDispense 系统的**电路**相关文件，包括电源管理、控制板、电机驱动、传感器与电路原理图等。

### 内容
- **Power（电源）**：电源方案、稳压与配电设计。
- **Control Board（控制板）**：用于控制点胶系统的 MCU 或 FPGA 相关设计。
- **Motor Driver（电机驱动）**：运动控制相关驱动电路，例如步进/伺服电机控制器。
- **Sensors（传感器）**：温控、运动反馈或视觉辅助等传感器设计。
- **Schematics（原理图）**：系统完整电气原理图。

### 目的
本目录用于提供 EDispense 平台所需的电子部件、电路设计与供电系统。

## English

### Introduction
This directory contains all the files related to the **electronic design** of the EDispense system, including power management, control boards, motor drivers, sensors, and circuit schematics.

### Contents
- **Power**: Power supply designs, voltage regulators, and distribution.
- **Control Board**: Microcontroller or FPGA-based designs for controlling the dispensing system.
- **Motor Driver**: Circuits and drivers for controlling motion, such as servo or stepper motor controllers.
- **Sensors**: Designs for sensors used in temperature control, motion feedback, or vision support.
- **Schematics**: Complete electrical schematics for the system.

### Purpose
The goal of this directory is to provide all necessary electronic components, circuit designs, and power systems required for the EDispense platform.
